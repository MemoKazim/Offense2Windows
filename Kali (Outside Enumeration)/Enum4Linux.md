>Enum4Linux is great tool for enumerating fundamental information from target machine

### Enumeration with verbose

``` zsh
enum4linux -v <IP>
```

### OS information

``` zsh
enum4linux -o <IP>
```

### List groups

``` zsh
enum4linux -G <IP>
```

### List Windows shares

``` zsh
enum4linux -S <IP>
```

### List user names

``` zsh
enum4linux -u <USERNAME> -p <PASSWORD> -U <IP> 
```
